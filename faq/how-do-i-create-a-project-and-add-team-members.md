# How do I create a project and add team members?

To create a project in AI/Run CodeMie, open **Settings → Administration → Projects management**
and click **Create**. Enter a unique, lowercase project name and an optional description, then
click **Create**. The project appears in the list immediately.

Once the project is created, open it and use **Add User to Project** to invite teammates one at
a time by searching for their name or email, or use **Import Users** to upload a CSV file with
multiple users at once. When adding users, choose their role — **User** for standard access or
**Project Admin** to allow them to manage project members and settings.
