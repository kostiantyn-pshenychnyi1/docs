# Can I change the LLM model for a single conversation without editing the assistant?

Yes. Click the **model selector** button in the chat input toolbar (it shows **Default**
when no override is active). A searchable panel lists all available models. Select any
model to apply it to subsequent messages in that conversation. To revert, choose
**Assistant Default** from the same panel.

The model override applies only to the current conversation and does not change the
assistant's configuration.
