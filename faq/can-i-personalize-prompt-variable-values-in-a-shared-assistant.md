# Can I personalize prompt variable values in a shared assistant?

Yes. When an assistant is shared with your project, you can set your own values for its prompt variables directly from the assistant's details page. Open the assistant, scroll to the **Prompt Variables** section, edit the values you need, and click **Save** next to each one.

Your values are stored per-user and applied automatically in your chat sessions. They do not affect the assistant's default values or other team members.
