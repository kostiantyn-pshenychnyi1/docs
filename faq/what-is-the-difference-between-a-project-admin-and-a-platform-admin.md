# What is the difference between a Project Admin and a Platform Admin?

A **Project Admin** is a user who has administrative rights within one or more specific
projects. Project Admins can add and remove project members, change their roles, and manage
project settings for the projects they own or administer. Any user can become a Project Admin
of a project they created or have been assigned as one.

A **Platform Admin** (Super Admin) has platform-wide access. In addition to everything a
Project Admin can do, a Platform Admin can view and manage all projects across the platform,
access the **Users Management** page to inspect all registered users, edit user names, and
manage any user's project assignments from a central panel.
