# How do I enable Web Search or Code Interpreter in a chat conversation?

Click the **sliders icon** in the chat input toolbar to open the **Tools Settings** panel.
Toggle **Web Search** on to let the assistant search the internet for up-to-date
information, or toggle **Code Interpreter** on to enable Python code execution for data
analysis and visualizations. Your settings are saved per conversation and restored
automatically when you return to the same chat.

These toggles only appear if an administrator has enabled the corresponding features for
your environment. If both are disabled, the sliders icon is hidden entirely.
