# How do I set a budget for a project in CodeMie?

Project budgets are managed by super-admins through **Settings → Administration → Projects Management**.

1. Select the project you want to budget.
2. In the **Budgets** section, find the category card you want (Platform, CLI, or Premium models) and click **Add Budget**.
3. Fill in the budget form: name, category, reset period, soft limit (warning threshold), and hard limit (enforcement cap in USD).
4. Click **Create**.

CodeMie automatically distributes the budget equally across all active project members and syncs the allocation with LiteLLM for enforcement. Each project can have up to three independent budgets — one per category.
