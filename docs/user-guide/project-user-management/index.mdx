# Project & User Management

Manage your projects and team members directly from the AI/Run CodeMie interface. Create and
configure projects, invite teammates, assign roles, and — for Platform Admins — oversee all
users and their project memberships across the entire platform.

The **Project & User Management** section is accessible via the **Profile** icon in the bottom-left corner → **Settings → Administration**.

## Access by Role

| Feature                       | Regular User (Project Admin) | Platform Admin |
| ----------------------------- | ---------------------------- | -------------- |
| Create a project              | ✅                           | ✅             |
| View own projects             | ✅                           | ✅             |
| Edit own project              | ✅                           | ✅             |

| Manage members in own project | ✅                           | ✅             |
| View and manage all projects  | ❌                           | ✅             |
